# 工作查询展现格式（钉钉移动端适配）

> Route B 输出模板，每个 section 按需展现，无内容则不列。
> 钉钉以手机阅读为主，**禁用表格**，改用缩进列表。

## 格式

```
**YYYY年M月D日（周X）工作安排**

**专项**
- {任务描述} — {责任人/组织人}

**每日必做**
{工作项} — {责任人}
{工作项} — {责任人}
...

**值班**
当日值班人员：{姓名}（参照"今日我值班工作提示卡"）

{其他 section（同上缩进列表格式）}
```

## Section 规则

| 展示名 | 触发条件 | 内容来源 |
|--------|----------|----------|
| 专项 | tasks.md 有当日待办 | tasks.md 非【固定】条目 |
| 每日必做 | 每天固定 | 工班日常固定工作.md → 每日 |
| 值班 | 每天 | 值班轮序推算 |
| 演练后续 | 演练当天 | 按需/触发式 → 个人演练档案 |
| 周五任务 | 当日是周五 | 工班日常固定工作.md → 每周五 |
| 月度任务 | 与当日日期匹配 | 工班日常固定工作.md → 每月固定日期 |

## 责任人表述规范

| 原始表述 | 展示用语 |
|----------|----------|
| 值班人员 | 当日值班人员 |
| 各库区负责人 | 各库区负责人（按分管区域） |
| 谁借钥匙谁填写 | 借用人 |
| 出入库/危废废旧岗位同事 | 出入库/危废废旧岗位人员 |
| 所有人(参演人员) | 全体参演人员 |
