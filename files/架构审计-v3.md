## 1. 架构概览

### 1.1 当前数据流（已验证）

```
  CLI 入口 (唯一活跃路径)
  ─────────────────────────────────────────────────
  entry.py → Pipeline.build_default().run()
              ├── layer1: ingress.build()
              │     └── query_router.classify() → route (event|task|profile|knowledge)
              ├── layer2: intent.extract()
              │     └── event.extract() + context.resolve() + record_manager
              ├── layer3: reasoning.reason()
              ├── layer4: execution.execute()
              ├── layer5: response.respond()
              │     └── task_handler / knowledge_handler
              └── layer6: reflection.reflect()
                    ├── Phase A: 观察提炼 (LLM, 3600s cooldown)
                    └── Phase B: CognitiveLoop
                          ├── Probe: LLM 分析+假设
                          └── Simulate: CausalSimulator 推演
```

**验证结论: entry.py 与 Pipeline 是同一路径**（非两条独立入口）。  
`pipeline.py` 无 `__main__`，纯库模块，仅被 entry.py 调用。
